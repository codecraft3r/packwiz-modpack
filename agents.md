## Future Update Rules

- Do not edit `pack.toml`, `index.toml`, or `*.pw.toml` files directly.
- Use packwiz commands for metadata changes, for example:

  ```sh
  packwiz curseforge add --addon-id <project-id> --file-id <file-id>
  packwiz remove
  packwiz update --all
  packwiz refresh
  ```

- To migrate to a newer official All The Mons release, download the new CurseForge archive and import it in a clean worktree or branch with `packwiz curseforge import`. Compare counts and generated metadata before replacing this repository state.
- Use `packwiz update --all` only when intentionally diverging from the exact official All The Mons CurseForge release.
- After any non-metadata file changes, run `packwiz refresh`.
- After publishing, replace every `REPLACE_ME` raw GitHub URL with the actual owner/repository path.
